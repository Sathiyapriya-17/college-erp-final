# Bibliography

The development and architectural design of the College ERP System were informed and supported by the official documentation, academic theories, and community resources associated with the specific technologies implemented.

1.  **Angular Developers & Google (2024).** *Angular Official Documentation (v18.0.0).* Retrieved from [https://angular.dev](https://angular.dev)
    *(Referenced for orchestrating Single Page Application (SPA) architecture, reactive forms, and RxJS observable state management.)*

2.  **PrimeTek Informatics (2024).** *PrimeNG UI Component Library Documentation.* Retrieved from [https://primeng.org](https://primeng.org)
    *(Referenced for the implementation of responsive structural components, data grids, and asynchronous toast messaging interfaces.)*

3.  **Microsoft Corporation (2024).** *TypeScript Language Specification.* Retrieved from [https://www.typescriptlang.org](https://www.typescriptlang.org)
    *(Referenced for ensuring strict static typing and interface compliance across the frontend Angular architecture.)*

4.  **Django Software Foundation (2024).** *Django (v5.x) Documentation.* Retrieved from [https://docs.djangoproject.com](https://docs.djangoproject.com)
    *(Referenced for Python backend routing, Object-Relational Mapping (ORM) design, cryptographic hashing algorithms, and built-in CSRF security protocols.)*

5.  **Encode OSS Ltd (2024).** *Django REST Framework (DRF) Documentation.* Retrieved from [https://www.django-rest-framework.org](https://www.django-rest-framework.org)
    *(Referenced for the serialization of complex Python data structures into lightweight JSON payloads and the creation of stateless RESTful API endpoints.)*

6.  **The PostgreSQL Global Development Group (2024).** *PostgreSQL Documentation.* Retrieved from [https://www.postgresql.org/docs](https://www.postgresql.org/docs)
    *(Referenced for establishing strict relational database schemas, enforcing composite data integrity constraints, and optimizing indexing for the attendance and examination data layers.)*

7.  **Mozilla Developer Network (MDN) (2024).** *Cross-Origin Resource Sharing (CORS) Web Technology Guidelines.* Retrieved from [https://developer.mozilla.org](https://developer.mozilla.org)
    *(Referenced for securely configuring the asynchronous data pipeline between the decoupled Django server port and the Angular client host.)*
